Алгоритм Манакера (также известный как алгоритм поиска палиндромов Манакера) является эффективным алгоритмом для нахождения всех палиндромов заданной строки за линейное время.

Основная идея алгоритма Манакера заключается в том, что он использует симметрию палиндромов, чтобы избежать повторных вычислений. Алгоритм строит массив, называемый "радиусами палиндромов", который для каждой позиции строки указывает расстояние от этой позиции до ближайшего палиндрома с центром в другой позиции.

Шаги алгоритма Манакера следующие:

Преобразование исходной строки для обработки случаев палиндромов нечетной длины: вставляем специальные символы "#" между символами и в начало и конец строки. Например, строка "abba" становится "#a#b#b#a#".
Инициализация радиусов палиндромов для каждой позиции. Начальные радиусы для всех позиций устанавливаются равными 0.
Обработка строки с использованием центра палиндрома "центрального элемента". Для каждого элемента строки:
Определение текущего палиндрома с использованием уже найденных палиндромов.
Обновление радиусов палиндромов для текущего элемента.
Нахождение наибольшего радиуса палиндрома и его центрального элемента, что дает самый длинный палиндром.
Построение списка всех найденных палиндромов с использованием радиусов исходной строки.
Алгоритм Манакера имеет линейную сложность O(n), где n - длина строки. Это делает его очень эффективным для нахождения всех палиндромов в строке.
